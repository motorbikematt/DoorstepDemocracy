# Doorstep Democracy Site Rules

(a) This repo is a public web root, no internal docs ever.
(b) Design-token law: tokens live in style.css :root; never hardcode hex.
(c) One h1 per page; every page ships title/meta-description/canonical/OG.

(e) Commit-don't-push workflow.
(f) Zero PII incl. realistic synthetic data.
(g) Nav must list the same links on every page.
(h) Jekyll coexistence note: plain .html files pass through the Pages Jekyll build untouched; blog lives under /blog/.
